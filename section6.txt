6.2 Usability and Output Clarity

Tool        TP  FP  FN  Accuracy
Splint      4   0   1   80%
Cppcheck    1   0   4   20%
Flawfinder  3   4   2   33%

Each static analysis tool was evaluated based on both detection performance and usability. Splint identified most of the known vulnerabilities, but its output included additional warnings that made it harder to interpret. Cppcheck produced simple and easy-to-read output, but it detected only one vulnerability, limiting its effectiveness. Flawfinder identified several key vulnerabilities and provided useful CWE references, but it also generated extra warnings, increasing false positives. Overall, tools that balance detection accuracy and clear output are more useful for developers.
